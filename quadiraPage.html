CustomerPortal

Nu ben ik vooral bezig met het werken in een portaal wat Quadira heeft gemaakt. Namelijk de CustomerPortal. Dit portaal is gemaakt zodat klanten bijvoorbeeld releases kunnen downloaden van Advanced Froms. Ik ben voor dit portaal tickets aan het oplossen. Dat doe ik door code toe te voegen / aan te passen in Vue en C#. Hier zal uiteindelijk een nieuwe release voor uitkomen waar ook tickets bijzitten die ik heb gemaakt.
